# Trouwwebsite Floris & Amber

Deze map bevat een complete statische trouwwebsite.

## Bestanden

- `index.html` – inhoud van de website
- `styles.css` – vormgeving
- `script.js` – aftelklok, animaties, mobiel menu en RSVP

## Gratis online zetten via GitHub Pages

1. Maak gratis een account op GitHub.
2. Maak een nieuwe openbare repository, bijvoorbeeld `floris-en-amber`.
3. Upload de drie bestanden uit deze map.
4. Open in GitHub: Settings > Pages.
5. Kies bij Source: `Deploy from a branch`.
6. Kies branch `main` en map `/root`.
7. Klik op Save.

Na enkele ogenblikken verschijnt de publieke link in hetzelfde scherm.

## RSVP e-mailadres aanpassen

Open `script.js` en zoek:

`florisenamber@example.com`

Vervang dit door het juiste e-mailadres.

## Teksten of foto's aanpassen

Teksten staan in `index.html`.

De grote foto's worden geladen vanaf Unsplash. Zoek in `styles.css` naar `background:url(...)` om ze te vervangen door eigen foto's of andere afbeeldingen.
